# curriculum-policy.md — 教材の学年・難易度ポリシー

## 基本方針

- 文科省の「学年別漢字配当表」を前提に、学年に対して難しすぎない範囲で出題する。
- 未習の漢字は、ひらがな表記にするか、ふりがなを添える。
- 問題文は短く、ひらがな多め。1問1論点。
- 「かなり簡単」から始める。最初のつまずきを直すのが目的で、選抜ではない。

## 漢字

- 小2ページ（grade2）＝ 小1漢字の復習 ＋ 小2漢字の基礎。
- 小3ページ（grade3）＝ 小2漢字の復習 ＋ 小3漢字の基礎。
- 出題漢字そのものは対象学年以下の配当を使う。
- 選択肢の読みや熟語に「その学年でまだ習わない漢字」を混ぜない（混ぜる場合はかな表記）。
- 書き順の完全再現はP0では扱わない。読み・意味・熟語・文中補充・書き取り欄を優先。

### 学年別配当の目安（本ドリルで使う範囲）

- 小1：一 二 三 forty…（80字）例：山 川 花 空 天 気 name 目 手 足 …
- 小2：里 come 教 …（160字）例：春 夏 秋 冬 朝 昼 morning 夜 元 気 …
- 小3：（200字）例：登 号 秒 practice 動 由 …
- 実データでは、各問の `grade` に「その漢字の配当学年」を入れる。
  復習問題は対象学年より下の配当漢字を使う。

（注：上記は運用の目安。実際の配当は文科省の配当表を都度確認する。）

## 国語

- 主語・述語・助詞（は/を/へ）・指示語（これ/それ/あれ）・3行以内の短文読解を扱う。
- 長文読解・比喩・要約・抽象的な心情読解はP0では扱わない。
- 例文は身近な生活場面（公園・学校・家・食事・季節）に限定する。

## 英語（スペル補修）

- 会話表現ではなく、アルファベット・音・スペルの対応を扱う。
- 大文字/小文字の対応、3文字単語、身近な単語を中心にする。
- 最初に扱う単語：cat / dog / sun / pen / bag / red / blue / book / apple / milk。
- 難しい文法・長文・英作文はP0では扱わない。

## 算数

- 維持用。たし算・ひき算・九九・短い文章題のみ。
- 小2・小3が「概ねできる」前提なので、ひねり問題は入れない。

## 監修時のチェック観点（curriculum-guard 用）

- 対象学年に対して語彙・漢字・文が難しすぎないか。
- 未習漢字にふりがな or かな表記があるか。
- 選択肢が紛らわしすぎないか（正解が一意に決まるか）。
- 解説が短く、子どもが読めるやさしさか。
