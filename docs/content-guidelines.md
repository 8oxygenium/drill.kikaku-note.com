# content-guidelines.md — 文言・トーンのガイドライン

## 基本トーン

- はげます・中立・やさしい日本語。
- 子どもを責めない。間違いは「つまずき」で、直せば良いものとして扱う。
- 保護者にも読みやすい、落ち着いた情報サイトの言葉づかい。

## 使ってよい言葉

- つまずき / やりなおし / 10分復習 / 基礎確認
- もう一度 / できたね / あと少し / ゆっくりでいい

## 使ってはいけない言葉

- 勉強できない子 / 落ちこぼれ / バカでもわかる / できない（断定）
- 「なんでできないの」など責める言い回し
- 競争・順位・煽り（「ライバルに差をつけろ」等）

## 正誤フィードバックの書き方

- 正解時：「せいかい！」など短く肯定。過度な演出はしない。
- 不正解時：「おしい」「もう一度みてみよう」。×の理由を短い解説で示す。
- 解説は1〜2文。まず答え、次に理由。むずかしい言葉を避ける。

例（漢字）：
> 春は「はる」と読みます。季節の名前です。

例（国語・助詞）：
> 「を」は、何かをする相手につきます。「ボールをける」。

## 表記

- 問題文はひらがな多め。学年で未習の漢字はかな or ふりがな。
- 句読点・記号は最小限。1問1論点。
- 数字は算数以外では原則かな or 少数の漢数字（一・二）にとどめる。

## デザイン上の禁止（文言に関わる範囲）

- キャラクターに子どもを煽らせない。
- 効果音・ガチャ風・課金風の表現を入れない。
- 広告前提の「続きはこちら」的な誘導を入れない。
